# クイックスタートガイド

## 簡単な起動方法

### ステップ1: 依存パッケージのインストール

```bash
cd /Users/hayato/financial_rpg
pip3 install -r requirements.txt
```

### ステップ2: サーバーを起動

```bash
python3 app.py
```

### ステップ3: ブラウザでアクセス

起動時に表示されるURLをコピーしてブラウザに貼り付けてください。

例:
- `http://localhost:8080`
- `http://127.0.0.1:8080`

## 重要なポイント

1. **サーバーを起動したままにしてください**
   - ターミナルウィンドウを閉じるとサーバーが停止します
   - サーバーを停止するには、ターミナルで `Ctrl+C` を押してください

2. **表示されたURLを正確に使用**
   - 起動時に表示されるポート番号（例: 8080）を使用してください
   - ポート5000が使用中の場合は、自動的に別のポートが使用されます

3. **ブラウザのアドレスバーに直接入力**
   - ブラウザの上部のアドレスバーに `http://localhost:8080` を入力してください
   - 検索エンジンで検索するのではなく、直接URLを入力することが重要です

## テスト方法

まずは簡単なテストサーバーで動作確認：

```bash
python3 test_server.py
```

ブラウザでアクセスして「✅ サーバーは正常に動作しています！」と表示されればOKです。

その後、本番サーバーを起動：

```bash
python3 app.py
```
